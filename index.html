Revolutionizing Wildlife Conservation

The ELOC (Elephant Locator) is an open-source, low-cost, solar-powered bioacoustic monitoring device specifically designed for elephant detection and wildlife conservation.

With its built-in ESP32 chip and advanced machine learning capabilities, ELOC offers enough processing power to run complex ML models for real-time sound recognition. The device continuously monitors the environment for elephant vocalizations and alerts conservationists when elephants are detected.

Our Mission

"Good bye disposable batteries" - Unlike existing recorders that dispose of more than 40 batteries per device every year, ELOC's solar-powered design eliminates battery waste while providing unlimited recording time.

Open Source Software + Open Source Hardware + DIY Motivation = Low Cost Open Source Device

Key Features

Advanced technology designed specifically for wildlife conservation and acoustic monitoring

Solar Powered

Unlimited recording time with solar panel and rechargeable batteries. No more disposable battery waste - environmentally friendly and cost-effective.

AI-Powered Detection

Real-time elephant sound recognition using machine learning models running on ESP32 chip. Continuous monitoring with instant alerts.

Full-Spectrum Audio

Records audio frequencies from 10 Hz to 22 kHz with sample rates up to 44kHz. Perfect for capturing elephant rumbles and other wildlife sounds.

Mobile App Control

Configure and control via Bluetooth with the ELOC Control Panel Android app. No PC required for field setup and configuration.

Open Source

Built on open-source principles with available hardware designs and software. Encourages collaboration and community customization.

Weather Resistant

IP66 rated enclosure designed for harsh environmental conditions. Reliable operation in diverse wildlife habitats.

Technical Specifications

Advanced hardware optimized for bioacoustic monitoring and wildlife research

Processing & Memory
  • Processor: ESP32-WROVER-E, Dual-core Xtensa LX6, 240MHz
  • Memory: 520KB SRAM + 8MB PSRAM
  • Flash: 16MB SPI Flash
  • Storage: MicroSD (up to 2TB, FAT32)
Audio Capabilities
  • Digital Mic: ICS-43434, 24-bit ADC
  • SNR: 65dBA, -26dBFS sensitivity
  • Frequency Range: 10Hz - 22kHz
  • Sample Rates: 8, 16, 24, 32, 44kHz
Power System
  • Input: Solar (4-18V) / USB-C
  • Battery: Li-Ion (3.7V) or LiFePo4 (3.2V)
  • Charger: BM2517M - max 800mA
  • Runtime: >30 days @16kHz recording
Additional Features
  • Sensors: LIS3DH Accelerometer, Temperature
  • Connectivity: Bluetooth, Wi-Fi
  • Expansion: LoRa, GPS, GSM modules
  • Interface: I2C, I2S, SPI, UART

Getting Started

Simple setup process to get your ELOC device recording in the field

1
What You Need
  • ELOC 3 device
  • MicroSD card (FAT32, up to 2TB)
  • Li-Ion or LiFePo4 battery
  • Solar panel (4-18V)
  • Android phone with ELOC app
2
Setup Process
  1. 1. Connect battery to ELOC
  2. 2. Install ELOC Control Panel app
  3. 3. Turn ELOC ON (wait for blue LED)
  4. 4. Connect via Bluetooth in app
  5. 5. Configure settings and start recording
3
Deployment
  • Position solar panel for optimal light
  • Mount securely in target habitat
  • Ensure good GPS/cellular coverage
  • Weather-resistant installation
